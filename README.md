# Github Notifier
==========
Github Notifications on your menu bar.

### Prerequisites

 - Electron
 - ReactJS
 - Grunt
 - NPM


### Installation
If you encounter any issues with `npm install`, then run `ulimit -n 512`.

    npm install


### Development
To watch for changes in the `src` directory:

    npm run watch

To run the actual **electron app**:

    npm start


### Tests (JsxHint)

    npm test
